🐛 Vue Router 调试工具

步骤 1: 打开应用

步骤 2: 在打开的页面中按 F12

然后在 Console 标签中**依次执行**以下命令：

① 检查当前路由

window.location.hash

应该显示: "#/admin/login"

② 检查 Vue Router 实例

document.querySelector('#app').__vueParentComponent?.appContext?.config?.globalProperties?.$router

或者:

window.__VUE_ROUTER__

③ 检查当前路由对象

const app = document.querySelector('#app').__vueParentComponent;
const router = app.appContext.config.globalProperties.$router;
console.log('当前路由:', router.currentRoute.value);
console.log('所有路由:', router.getRoutes());

④ 手动导航测试

const router = document.querySelector('#app').__vueParentComponent.appContext.config.globalProperties.$router;
router.push('/admin/login');

⑤ 查看路由视图

document.querySelector('router-view') || document.querySelector('[data-v-app]').innerHTML
