🔐 认证诊断工具

使用说明

这个工具帮助诊断前端认证问题。请按顺序执行以下步骤：

步骤 1: 检查当前Token状态

步骤 2: 快速登录测试

如果没有token或token无效，可以使用测试账号快速登录：

步骤 3: 测试API连接

步骤 4: 手动执行诊断命令

在浏览器控制台(F12)中依次运行以下命令： 
// 1. 检查所有token相关项
Object.keys(localStorage).filter(k => k.includes('token') || k.includes('Token')).forEach(k => console.log(k, ':', localStorage.getItem(k)))

// 2. 检查特定token
console.log('access_token:', localStorage.getItem('access_token'))
console.log('token:', localStorage.getItem('token'))
console.log('admin_token:', localStorage.getItem('admin_token'))
console.log('user_token:', localStorage.getItem('user_token'))

// 3. 测试API调用
fetch('/api/v1/beidan-filter/strategies', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + (localStorage.getItem('access_token') || localStorage.getItem('token') || '')
  }
}).then(r => r.json()).then(console.log).catch(console.error)
