## 🔍 技术细节展示

### 后端处理流程

1. **读取数据文件**
```python
# 从 debug/500_com_matches_*.json 读取
with open(file_path, 'r', encoding='utf-8') as f:
    matches = json.load(f)
```

2. **筛选周一比赛**
```python
monday_matches = [m for m in matches if m.get('match_id', '').startswith('周一')]
```

3. **计算热度**
```python
def _calculate_popularity(match):
    popularity = 50  # 基础热度
    # 根据联赛加权
    popularity += league_weights.get(league, 10)
    # 根据赔率方差调整
    return min(max(popularity, 0), 100)
```

4. **格式转换**
```python
formatted_match = {
    "id": match.get("match_id"),
    "league": match.get("league"),
    "home_team": match.get("home_team"),
    "away_team": match.get("away_team"),
    "match_time": match.get("match_time"),
    "match_date": _parse_match_date(match.get("match_time")),
    "odds_draw": match.get("odds_draw", 0),
    "popularity": _calculate_popularity(match),
    "source": "500彩票网"
}
```

### 前端处理流程

1. **API调用**
```javascript
const response = await getJczqMatches({
  days: 3,
  sort_by: 'date',
  source: '500'
});
```

2. **数据绑定**
```vue
<div v-for="match in paginatedMatches" :key="match.id">
  <div class="match-item">
    {{ match.home_team }} VS {{ match.away_team }}
  </div>
</div>
```

3. **响应式更新**
```javascript
const matches = ref([])  // 自动触发视图更新
matches.value = response.data
```
