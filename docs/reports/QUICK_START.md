## 🔄 如果页面仍然空白

### 刷新浏览器
1. 按 `F5` 刷新
2. 或按 `Ctrl + F5` 强制刷新（清除缓存）

### 检查浏览器控制台
1. 按 `F12` 打开开发者工具
2. 切换到 **Console** 标签
3. 查看是否有错误信息

### 等待后端完全启动
后端服务器可能需要10-30秒完全启动，请耐心等待后再刷新页面

## ⚡ 快速命令参考

### 重启前端
```bash
cd frontend && npm run dev
```

### 重启后端
```bash
python -m uvicorn backend.main:app --port 8000 --reload
```

### 运行爬虫更新数据
```bash
python crawl_500_com.py
```

### 一键启动全部
```bash
start_full_stack.bat
```
