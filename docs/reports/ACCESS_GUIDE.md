## 🔍 为什么端口是3005？

1. **配置端口**: vite.config.js 配置的是 `3000`
2. **实际端口**: 但3000可能被占用，系统自动选择了 `3005`
3. **Vite行为**: 如果配置端口被占用，Vite会自动使用下一个可用端口

## 🛠️ 如果页面仍然无法显示

### 问题1: 页面空白
**解决方案**:
- 按 `Ctrl + F5` 强制刷新
- 清除浏览器缓存
- 确认使用 `#/jczq-schedule` 而不是 `/jczq-schedule`

### 问题2: 显示"暂无数据"
**解决方案**:
- 确认后端正在运行（访问 http://localhost:8000/docs）
- 检查API是否返回数据（访问 http://localhost:8000/api/v1/jczq/matches?source=500）
- 查看浏览器控制台的网络请求

### 问题3: 404错误
**解决方案**:
- 确认使用正确的端口号（3005而不是3000或5173）
- 确认前端服务器正在运行
- 使用hash路由格式（带 `#/`）
