## 1. 目标与原则

- 目标：先解决发布阻塞（P0），再提升命中质量（P1），最后补齐运营能力（P2）。
- 原则：每个工单必须具备 `负责人角色 / 人天 / 依赖 / 交付物 / 验收标准`。
- 验收方式：接口联调 + 页面验证 + Playwright 回归三重通过。
