## 🔍 API端点详情

### 获取竞彩足球比赛

**端点**: `GET /api/v1/jczq/matches`

**参数**:
- `source` (string): `500` - 使用500彩票网数据
- `days` (int): `1` - 筛选周一比赛
- `league` (string, 可选): 联赛名称
- `sort_by` (string): `date`/`popularity`

**示例请求**:
```
http://localhost:8000/api/v1/jczq/matches?source=500&days=1
```

**响应示例**:

## 🐛 常见问题

### Q1: 后端启动失败？
**A**: 检查Python版本（>=3.8）和依赖安装
```bash
pip install -r requirements.txt
```

### Q2: 前端无法获取数据？
**A**: 检查：
1. 后端是否运行在8000端口
2. `.env.development` 配置是否正确
3. 浏览器控制台的错误信息

### Q3: 数据文件不存在？
**A**: 运行爬虫生成数据
```bash
python crawl_500_com.py
```

### Q4: 显示空数据？
**A**: 检查：
1. `debug/` 目录下是否有数据文件
2. API响应是否包含数据
3. 前端是否正确解析数据
