## 🆘 故障排查

### 常见问题 TOP 5

#### 1. 导入错误: ModuleNotFoundError

**原因**: 文件移动后导入路径未更新

**解决**:
```bash
# 查找需要更新的文件
findstr /s /i "from backend.debug_" backend\*.py
findstr /s /i "from backend.get_" backend\*.py

# 更新导入路径
# 旧: from backend.debug_scraper import X
# 新: from backend.debug.debug_scraper import X
```

#### 2. 前端 Pinia Store 找不到

**原因**: stores 路径更新

**解决**:
```javascript
// 更新导入
// 旧: import X from '@/components/store/modules/X'
// 新: import X from '@/stores/modules/X'
```

#### 3. 测试失败

**原因**: 测试文件导入路径未更新

**解决**: 更新测试文件中的导入路径

#### 4. Git 冲突

**原因**: 其他成员同时修改

**解决**: 使用 `git stash` 暂存，解决冲突后 `git stash pop`

#### 5. 性能下降

**原因**: 可能的导入循环

**解决**: 检查新的导入关系，避免循环依赖
