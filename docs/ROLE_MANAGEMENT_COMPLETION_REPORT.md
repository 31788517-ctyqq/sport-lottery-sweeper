## 🔧 技术实现细节

### 数据库字段
```sql
ALTER TABLE roles ADD COLUMN level INT DEFAULT 1;
ALTER TABLE roles ADD COLUMN is_system BOOLEAN DEFAULT FALSE;
```

### 权限编码体系
```
admin:user:view          -- 用户系统/查看用户
admin:role:edit          -- 角色权限/编辑权限
datasource:create        -- 数据源/创建
task:execute             -- 任务/执行
... (100个权限)
```

### API端点
```
GET  /api/v1/roles/              -- 获取所有角色
POST /api/v1/roles/              -- 创建角色
PUT  /api/v1/roles/{id}          -- 更新角色
DELETE /api/v1/roles/{id}        -- 删除角色

GET  /api/v1/permissions/        -- 获取权限列表
GET  /api/v1/permissions/tree    -- 获取权限树
GET  /api/v1/permissions/flat    -- 获取平组权限
```
