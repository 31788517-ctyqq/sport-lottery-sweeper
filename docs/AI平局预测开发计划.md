# AI平局预测 开发计划（审批版）

## 1. 目标与范围
**目标**：新增“AI平局预测”功能，对当天北单比赛进行平局概率扫盘，形成推荐排名，并支持模型数据详情查看。
**范围**：
- 前端新增“AI平局预测”页面（北单当日扫盘列表 + 详情弹窗）
- 后端新增“AI平局预测”模型与当日扫盘接口
- 数据源限定为“盈球”北单数据
- 后台菜单：平局预测管理 → 新增“AI平局预测”子菜单

## 2. 需求拆解（与用户条目对齐）
1) **当天北单平局扫盘页面**：展示当日北单比赛列表、平局概率、推荐排名、筛选与分页。
2) **平局预测模型**：基于“docs/AI平局预测简化公式（实战版）.md”实现模型，完成当日扫盘。
3) **数据来源**：北单比赛数据仅使用“盈球”数据，字段与赔率以该来源为准。
4) **菜单新增**：后台管理系统“平局预测管理”下新增“AI平局预测”。
5) **先计划后开发**：本计划文档审批后进入实现阶段。

## 3. 数据源与采集策略（盈球）
**数据来源**：
- 仅使用“盈球”北单比赛数据（含当日赛程与赔率信息）。

**模型所需数据类型**（按优先级）：
- 必须：胜平负赔率（主胜/平/客胜）
- 必须：大小球盘口（用于小球环境指数）
- 可选：让球盘口/附加盘口（用于补充强弱判断或校验）
- 可选：赛程类型/阶段信息（用于战意指数 D 的人工/规则赋值）

**数据缺失处理策略**（规划）：
- 若“盈球”缺少大小球盘口：
  - 方案A：使用平赔结构推断小球环境指数 C（后端规则）
  - 方案B：允许 C 缺省并标记为“缺失”，模型仍可输出，但评级降权
- 若赔率缺失：该场次不参与扫盘并在前端提示原因

## 4. 模型方案（AI平局预测）
**核心公式**（来自文档）：
\[
P(\text{Draw}) = 0.248 + 0.32A + 0.28C + 0.24B + 0.16D
\]

**各指数定义**（数据校准版）：
- **A 实力接近指数**：
  \[
  A = e^{-3\left| \ln\left(\frac{主胜}{客胜}\right) \right|}
  \]
- **B 平赔结构指数**：
  \[
  B = e^{-\left(\frac{平赔-3.05}{0.45}\right)^2}
  \]
- **C 小球环境指数**：
  \[
  C = e^{-0.9(大小球-2.2)}
  \]
  并限制在 0–1
- **D 战意指数**：
  - 默认 0（普通联赛）
  - 可扩展为规则或人工赋值（+0.6 / +0.4 / -0.6 等）

**概率解释**：
- P ≥ 0.36：高平
- 0.31–0.35：偏高
- 0.26–0.30：正常
- 0.22–0.25：偏低
- < 0.22：不推荐

## 5. 数据落库与接口设计（规划）
**表/模型（建议）**：
- `ai_draw_prediction_sources`：原始赔率与赛程快照
- `ai_draw_prediction_features`：A/B/C/D 与中间量
- `ai_draw_prediction_results`：最终 P(draw) 与排名

**后端 API（建议）**：
- `POST /api/v1/draw-prediction/ai-draw/fetch`（手动抓取/计算）
- `GET /api/v1/draw-prediction/ai-draw/list?date=YYYY-MM-DD&data_source=yingqiu`
- `GET /api/v1/draw-prediction/ai-draw/detail/{match_id}`

## 6. 前端页面与交互（规划）
**页面路径**：`/admin/draw-prediction/ai-draw`（待确认）

**列表字段（建议）**：
- 编号、比赛时间、联赛、主客队
- 主胜/平/客胜赔率
- 大小球盘口
- A/B/C/D 指数（可选展示）
- 平局概率 P(draw)
- 推荐等级（高平/偏高/正常/偏低）
- 排名

**操作**：
- “手动抓取并计算”按钮
- “模型数据”弹窗：展示四类数据取值与中间量

**模型数据弹窗四类数据**（与文档一致）：
1) 实力接近指数 A（来自主胜/客胜赔率）
2) 平赔结构指数 B（来自平赔）
3) 小球环境指数 C（来自大小球盘口）
4) 战意指数 D（默认 0，可扩展）

**空值规则**：任一字段缺失时显示 “-”，并在弹窗标记为“缺失”。
