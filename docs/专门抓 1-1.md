# 🎯 1-1 专用 Poisson 模型（核心）

Poisson比分模型的核心是：

* 主队进球 (X \sim Poisson(\lambda_H))
* 客队进球 (Y \sim Poisson(\lambda_A))

那么：

[
P(1\text{-}1)=P(X=1)\cdot P(Y=1)=\big(\lambda_H e^{-\lambda_H}\big)\cdot\big(\lambda_A e^{-\lambda_A}\big)
]

**你要做的只有一件事：把 (\lambda_H,\lambda_A) 估得更准、更“偏向1-1”。**

---

# 🧠 一、先用盘口把总进球和强弱拆出来（最实用）

你通常最容易拿到的是：**大小球线**（O/U）和**亚洲让球**（AH）。

我们用它们来构造“市场隐含进球均值”，非常实战。

## 1) 总进球均值 (\mu_T)（从大小球得到）

用一个简单映射（够用且稳定）：

| 大小球线 | (\mu_T) 建议值 |
| ---- | ----------- |
| 2.0  | 2.05        |
| 2.25 | 2.25        |
| 2.5  | 2.45        |
| 2.75 | 2.70        |
| 3.0  | 2.95        |

> 你也可以更精细：按水位微调（大球低水 → (\mu_T)+0.05；小球低水 → (\mu_T)-0.05）

---

## 2) 进球差均值 (\mu_D)（从让球得到）

用“让球 → 预期净胜球”的近似（简化但好用）：

| 让球（主让为正）      | (\mu_D) |
| ------------- | ------- |
| 0             | 0.00    |
| -0.25 / +0.25 | ±0.20   |
| -0.5 / +0.5   | ±0.38   |
| -0.75 / +0.75 | ±0.55   |
| -1.0 / +1.0   | ±0.72   |

---

## 3) 拆出 (\lambda_H,\lambda_A)

[
\lambda_H = \frac{\mu_T+\mu_D}{2},\quad
\lambda_A = \frac{\mu_T-\mu_D}{2}
]

到这里你已经有了基础版 Poisson。

---

# 🗡️ 二、“杀手级”增强：专门让模型更擅长 1-1

1-1 最常见的环境是：

* 总进球中等偏低（≈2.1–2.5）
* 强弱差不大（净胜球≈0）
* 两队“都有球但不多”（(\lambda)接近1）

所以我们加两步增强：**收缩 + 1-1形状增益**。

---

## ✅ 增强1：收缩（Shrinkage）到“1球区间”

当数据或盘口噪声较大时，把 (\lambda) 往 1.05 收一点（1-1的甜点区）。

[
\lambda'_H = (1-s)\lambda_H + s\cdot 1.05
]
[
\lambda'_A = (1-s)\lambda_A + s\cdot 1.05
]

推荐收缩系数 (s)（只要看两点：大小球 + 让球）：

* 若 **AH=0 或 ±0.25** 且 **O/U ≤2.5** → (s=0.20)
* 若 **AH=±0.5** 或 **O/U=2.75** → (s=0.10)
* 其他 → (s=0.00)

> 这一步会让模型在“典型平局盘”对 1-1 更敏感，回测里通常更稳。

---

## ✅ 增强2：相关性修正（Dixon–Coles 的简化版，专攻低比分）

经典 Poisson 假设主客进球独立，但足球低比分（0-0、1-1）常有相关性。
我们给 1-1 一个修正系数 (\tau)：

[
P_{DC}(1\text{-}1)=P_{Pois}(1\text{-}1)\cdot (1+\rho)
]

推荐 (\rho) 的经验设置（专门对“闷平盘”）：

* 若 **O/U ≤2.25** 且 **AH ∈ {0, ±0.25}** → (\rho=+0.10)
* 若 **O/U=2.5** 且 **AH接近0** → (\rho=+0.06)
* 其他 → (\rho=0)

> 直觉：小球+均势时，1-1会被低估一点点。

---

# 🧮 三、最终 1-1 概率公式（你就用这个）

1. 先算 (\mu_T,\mu_D)
2. 拆出 (\lambda_H,\lambda_A)
3. 做收缩得到 (\lambda'_H,\lambda'_A)
4. 算泊松 1-1 概率，再乘相关修正

[
P(1\text{-}1)=\Big(\lambda'_H e^{-\lambda'_H}\Big)\Big(\lambda'_A e^{-\lambda'_A}\Big)\cdot (1+\rho)
]

---

# 📊 四、Excel 直接可用写法（无宏）

假设你在表里输入：

* B2 = 大小球线（2, 2.25, 2.5, 2.75, 3.0）
* C2 = 亚洲让球（主让为负/正你自行统一；这里用“主让为正”表示主队强）
* D2 = （可选）大/小球水位倾向（例如 “O”/“U”，也可以不填）

### 1) (\mu_T)（大小球 → 总进球均值）

用近似映射（示例用IFS）：

```excel
=IFS(B2=2,2.05,B2=2.25,2.25,B2=2.5,2.45,B2=2.75,2.7,B2=3,2.95,TRUE,2.45)
```

### 2) (\mu_D)（让球 → 净胜球均值）

```excel
=IFS(C2=0,0,ABS(C2)=0.25,SIGN(C2)*0.2,ABS(C2)=0.5,SIGN(C2)*0.38,ABS(C2)=0.75,SIGN(C2)*0.55,ABS(C2)=1,SIGN(C2)*0.72,TRUE,SIGN(C2)*0.9*ABS(C2))
```

### 3) 拆 (\lambda_H,\lambda_A)

```excel
lambdaH = (muT + muD)/2
lambdaA = (muT - muD)/2
```

### 4) 收缩系数 s

```excel
=IF(AND(ABS(C2)<=0.25,B2<=2.5),0.2,IF(OR(ABS(C2)=0.5,B2=2.75),0.1,0))
```

### 5) 收缩后的 (\lambda')

```excel
lambdaH2 = (1-s)*lambdaH + s*1.05
lambdaA2 = (1-s)*lambdaA + s*1.05
```

### 6) 相关修正 rho

```excel
=IF(AND(B2<=2.25,ABS(C2)<=0.25),0.1,IF(AND(B2=2.5,ABS(C2)<=0.25),0.06,0))
```

### 7) 最终 1-1 概率

```excel
=(lambdaH2*EXP(-lambdaH2))*(lambdaA2*EXP(-lambdaA2))*(1+rho)
```

# 🧪 五、怎么用它“赚钱思路”（关键不是命中，是价值）

你用模型算出 (P(1-1)) 后，跟“1-1比分赔率”比较：

* 赔率 = Odds_11
* 隐含概率 (P_{imp}=1/Odds)

如果：

[
P(1\text{-}1) > \frac{1}{Odds_{11}} \times (1+\text{margin})
]

margin 你可以先取 8%（0.08）做安全垫。

# ✅ 六、1-1 的“甜点区”快速过滤（配合模型更强）

如果你只想先筛候选，命中下面越多越好：

* O/U 2.0–2.5
* AH 0 或 ±0.25
* 平赔在 2.8–3.1
* 双方都有进球倾向但不强（BTTS偏高，小球也偏高时尤其像1-1）
* 临场没有明显强热一边
