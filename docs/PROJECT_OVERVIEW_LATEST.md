# 体育彩票扫盘系统 - 项目概述（最新版）

## 项目简介

这是一个专业的竞彩足球数据采集、分析和预测平台，为竞彩足球提供全面的数据支持与智能决策辅助。系统结合了传统的数据采集分析能力与现代AI技术，包括大语言模型(LLM)、智能体系统等。

## 核心功能

### 1. 数据采集
- **多源数据爬取**：支持多个博彩公司和数据源的自动化数据采集
- **动态页面处理**：使用Playwright处理JavaScript渲染的页面
- **反爬虫对策**：实现IP池系统、请求头轮换等反反爬虫机制

### 2. 数据分析与预测
- **传统ML模型**：基于历史数据的统计分析和机器学习预测
- **LLM集成**：集成OpenAI、Gemini、Qwen等大语言模型
- **智能解释**：提供预测结果的自然语言解释

### 3. 智能体系统
- **自主决策智能体**：自动监控赔率变化并执行对冲策略
- **个性化推荐智能体**：根据用户画像提供定制化建议
- **协作式预测网络**：多个专业智能体协作完成复杂预测任务

### 4. 对冲策略
- **实时风险控制**：监控市场变化并执行对冲操作
- **资金管理**：智能分配资金以最大化收益并控制风险

### 5. 情报分析
- **多渠道情报收集**：整合社交媒体、新闻等外部信息
- **情感分析**：分析球队和球员相关的舆论情绪

## 技术架构

### 前端技术栈
- **Vue 3**：渐进式JavaScript框架
- **TypeScript**：静态类型检查
- **Tailwind CSS**：实用优先的CSS框架
- **Vite**：现代化的前端构建工具

### 后端技术栈
- **Python 3.11**：主要编程语言
- **FastAPI**：高性能Web框架，支持异步处理
- **SQLAlchemy**：ORM框架
- **Celery**：分布式任务队列
- **Redis**：缓存和消息队列
- **PostgreSQL**：主数据库（生产环境）/ SQLite（开发环境）

### AI/ML技术栈
- **OpenAI API**：GPT系列模型
- **Google Gemini**：Google的大语言模型
- **通义千问**：阿里巴巴的大语言模型
- **scikit-learn**：传统机器学习算法
- **PyTorch/TensorFlow**：深度学习框架（未来扩展）

## 系统模块

### 1. 数据采集模块
- **Scrapy爬虫框架**：高效的数据采集
- **Playwright**：处理JavaScript渲染的页面
- **IP池系统**：绕过反爬虫机制
- **数据清洗**：标准化不同来源的数据格式

### 2. 预测分析模块
- **机器学习模型**：基于历史数据的预测
- **LLM集成层**：统一的大型语言模型接口
- **预测解释器**：将模型输出转化为人类可理解的解释
- **风险评估**：预测结果的置信度和风险评估

### 3. 智能体系统
- **智能体基类**：统一的智能体接口
- **通信协议**：智能体间的消息传递机制
- **任务协调**：多智能体协作完成复杂任务
- **决策引擎**：基于多源信息的智能决策

### 4. 用户管理模块
- **JWT认证**：安全的身份验证
- **角色权限控制**：细粒度的权限管理
- **用户画像**：基于行为数据的用户建模

### 5. 对冲管理模块
- **实时监控**：持续监控市场变化
- **策略执行**：自动化执行对冲策略
- **风险控制**：限制潜在损失

## 开发指南

### 环境搭建
```bash
# 安装Python依赖
pip install -r requirements.txt

# 启动后端服务
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

# 启动前端服务
cd frontend && pnpm run dev
```

### 配置LLM
```bash
# 在.env文件中配置API密钥
OPENAI_API_KEY=your_openai_api_key
GEMINI_API_KEY=your_gemini_api_key
QWEN_API_KEY=your_qwen_api_key
```

### Docker部署
```bash
# 构建并启动服务
docker-compose up -d --build
```
